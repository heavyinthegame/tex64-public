# FAQ

## What is TeX64?

TeX64 is a LaTeX editor for writing and editing technical documents with a faster, more guided workflow.

## Does this repository contain the app source code?

No. This repository is only for public-facing product information and feedback.

## What platforms are supported right now?

The current public release target is macOS beta.

## Does TeX64 work with standard LaTeX?

Yes. TeX64 is designed to work with standard TeX workflows.

## Where can I download it?
